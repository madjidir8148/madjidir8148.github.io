HTML Reference Page

HTML Elements Description
HTML The HTML tag tells the browser that this is an HTML document. This tag represents the root of an HTML document.
Head The Header element represents a container for introductory content or a set of navigational links.
Title The Title tag is required in all HTML documents and it defines the title of the document.
Body The Body element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.
Heading The Heading element can include a title for the document, scripts, styles, meta information, and more.
Paragraph The Para tag is used to define parameters for plugins embedded with an object element.
Break The Break tag is an empty tag which means that it has no end tag.
Line The Line element is used to separate content or define a change in an HTML page.
Comment The Comment tag is used to insert comments in the source code.
Link The Link tag is used to link to external style sheets.
Image The Image tag creates a holding space for the referenced image.
List The Lists tag is used in ordered lists(ol), unordered lists (ul), and in menu lists (menu).
Table The Table element defines a table row, the element defines a table header, and the element defines a table cell.
Blocks The Block tag specifies a section that is quoted from another source.
